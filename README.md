## 📌 About

This repository contains all tasks completed during my **Web Development Internship** at **Cognifyz Technologies**. The internship is structured across **3 progressive levels**, each focusing on different aspects of front-end web development.

## ✅ Tasks Completed

### 🔵 Level 1 — Task 1: HTML & CSS Basics

| Feature | Description |
|---|---|
| Simple Webpage | Heading, paragraph, and image using semantic HTML |
| Basic Form | Single input field with a styled submit button |
| Navigation Menu | Three-link nav menu with hover and active states |

**Files:** `index.html`, `style.css`

---

### 🔵 Level 1 — Task 2: JavaScript

| Feature | Description |
|---|---|
| Color Change Button | Button cycles through 7 colors on each click |
| Greeting Alert | Displays morning / afternoon / evening / night greeting based on current time |
| Basic Calculator | Adds two numbers with input validation |

**Files:** `button.html/css/js`, `greeting.html/css/js`, `calculator.html/css/js`

---

### 🟣 Level 2 — Task 1: Front-end Frameworks

| Feature | Framework | Description |
|---|---|---|
| Responsive Card Component | Bulma | 3 cards with image, tag, title, description — responsive columns |
| Responsive Grid Layout | Bootstrap 5 | Hero + features + stats + sidebar/content layout |

**Files:** `bulma_card.html`, `bootstrap_grid.html`

---

### 🟣 Level 2 — Task 2: Responsive Design

| Feature | Description |
|---|---|
| Responsive Webpage | Adapts layout across desktop, tablet, and mobile using CSS media queries |
| Hamburger Menu | Mobile nav collapses into animated hamburger — toggles with JavaScript |

**Breakpoints used:**
- `> 900px` — Desktop: 4-column grid, side-by-side contact
- `≤ 900px` — Tablet: 2-column grid, stacked contact
- `≤ 600px` — Mobile: 1-column, hamburger menu visible

**Files:** `responsive.html`, `responsive.css`, `responsive.js`

---

## 🛠️ Technologies Used

- **HTML5** — Semantic structure
- **CSS3** — Styling, Flexbox, Grid, Media Queries
- **JavaScript (Vanilla)** — DOM manipulation, event handling
- **Bulma CSS** — Card component (CDN)
- **Bootstrap 5** — Grid layout (CDN)

## 🚀 How to Run

1. Clone this repository:
   ```bash

   ```

2. Open any `.html` file directly in your browser — no build tools or server needed.

3. For tasks using Bulma or Bootstrap, make sure you have an internet connection (CDN required).
